# MergeSort

MergeSort provides a recursive solution to sort the numbers given.

## Usage

1. Clone the repo
```bash
git clone https://github.com/JKanaiya/mergeSort
cd mergeSort/MergeSort
```
2. Install dependencies
```bash
npm install
```

4. Sort numbers given 
```bash
# Give the numbers eg node mergeSort.js 1,4,5,3,6,1
node mergeSort.js num1,num2,num3 ... num
```


Features
--------

- Displays the sorted numbers in an array
